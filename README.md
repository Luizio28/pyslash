# Pyslash
Um jogo ascii roguelike inspirado em dwarf fortress

# Rodando
Em linux:
```
python3 __main__.py
```

Em windows:
```
py __main__.py
```

# Dependências
- Colorama

```
pip install colorama
```

# Como jogar
- **Setas**: Mover
- **WASD**: Ação (Construir, Destruir)
- **1, 2, 3, 4**: Selecionar Itens

O cursor piscando representa o seu personagem.

Você pode regenerar o nível com Ctrl+N, carregar um nível salvo com Ctrl-L e salvar o nível atual com Ctrl-S.

Itens:
1- Parede de Pedra (=)
2- Tábua de Madeira (=)
3- Piso de Madeira (+)
4- Porta (Ω)
